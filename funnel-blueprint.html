Tech Stack (recommended, swappable)

  • CMS / Course host: Kajabi (preferred - all-in-one) or Teachable
  • Landing pages: Webflow (control) or Kajabi (speed)
  • Email: Kit / ConvertKit (preferred) or Kajabi-native
  • Quiz / Scorecard: ScoreApp (preferred for scoring + segmentation) or Typeform
  • Payment: Stripe (course) + Whop (mastermind / membership if cohort expands)
  • Video host: Wistia (course) + YouTube (long-form) + Loom (1:1 personal video reviews)
  • Analytics: GA4 + Meta Pixel + Microsoft Clarity (heatmaps)
  • CRM: GHL or Kajabi-native
  • Community / Mastermind: Skool (preferred - low overhead) or Circle
  • Scheduling: Calendly (default) or Cal.com
  • Project / build: Notion (handover-friendly) + Linear (engineering)

Integration Points

  1. Scorecard → Email: ScoreApp triggers Kit on quiz completion with score tag
  2. Email → Course: Kit triggers Kajabi enrolment on purchase webhook
  3. Course → Mastermind application: Course graduates routed to mastermind apply form on completion of module 6
  4. Podcast → Scorecard: episode mid-roll CTA routes listener to scorecard landing
  5. All events → GA4 + Meta Pixel via GTM: captures full funnel attribution
  6. CRM → Email: GHL syncs with Kit so 1:1 conversations land in the same record

Owner Map

  • Nick: filming (VSL + photoshoot), podcast mid-roll CTA recording, monthly Q&A sessions on the live course, mastermind facilitation, top-line decisions on copy + brand
  • AG1 (Andrei): everything else - landing page build, email writing, automation setup, content production cycles, ads management, scorecard build, weekly reporting

Failure Modes + Mitigations

  • VSL filming delays → use existing YouTube content as Phase 1 placeholder; replace with custom VSL by end of Phase 1
  • Course content creation timeline → outline modules first, record minimum viable version (60 min per module), add B-roll and polish after launch
  • Scorecard over-segmentation → start with 3 tiers; expand to 5 only after 1,000 quiz completions confirms volume by tier
  • Mastermind cohort fill timing → launch waitlist before the cohort opens; 40-person cap keeps quality of Q&A high

KPIs (90-day)

  • Phase 1: Custom Builder's Playbook landing live + 50+ buyers in first 30 days
  • Phase 2: 2,000+ scorecard completions + 8-12% IG-to-email conversion
  • Phase 3: 40+ short-form posts shipped + podcast CTA integration live
  • Phase 4: Meta CPL under $45 + ROAS 3x by week 12
  • Year-1 floor target: $300-750K incremental from course + email + mastermind cohort
